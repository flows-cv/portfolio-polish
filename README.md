# Portfolio Polish

Brutally honest portfolio audits, benchmarked against the best designers and engineers in the industry.

## Why this exists

We review thousands of portfolios placing designers and engineers at top companies. The same problems show up constantly. Meanwhile, the candidates who actually land roles at Linear, Vercel, Stripe, and Figma are doing the opposite of conventional portfolio advice: radical minimalism, craft pages, one-sentence descriptions, zero testimonials. We built this skill to close that gap so candidates stop optimizing for blog post advice and start optimizing for what actually works.

## What it evaluates

8 categories, each scored 1-5. Total out of 40.

- **10-Second Gut Check** (Critical) — typography, spacing, color, overall craft
- **Work Is the Hero** (High) — real work visible immediately, not process text
- **Ruthless Curation** (High) — every project earns its place
- **Ambition & Scope** (High) — obsessive craft, side projects, initiative
- **Storytelling & Framing** (Medium) — if text exists, is it good? typos kill.
- **Soul & Uniqueness** (Medium) — does this feel like a person or a template?
- **Portfolio as Product** (Medium) — IA, performance, responsiveness
- **Builder Signals** (Medium) — shipped tools, live demos, code

## Benchmarked against 28 reference portfolios

Including rauno.me, paco.me, emilkowal.ski, shud.in, benji.org, liumichelle.com, arjunmahesh.com, hector.me, and more. The skill cites specific portfolios in its feedback so candidates know exactly what to study.
